## 故事简介

「爱因斯坦垄断了互联网！必须打倒爱因斯坦！」    

随着床垫的吱吱响声，金发的女孩子蹭地站了起来，像自由女神一样比划着高举火炬的动作。  

我一边鼓掌一边附和着。
